# 想撤销最近一次提交，改怎么做呢

可以使用`git reset`命令

```shell
git reset HEAD^
```

命令的几个选项

1. `--mixed`

- **保留**工作区中的修改
- **撤销**暂存区中的修改（撤销git add）
- **撤销**版本库中的修改（撤销commit）

> 这个为默认参数，和不带参数效果相同

2. `--soft`

- **保留**工作区中的修改
- **保留**暂存区中的修改（保留git add）
- **撤销**版本库中的修改（撤销commit）

3. `--hard`

- **撤销**工作区中的修改
- **撤销**暂存区中的修改（撤销git add）
- **撤销**版本库中的修改（撤销commit）
